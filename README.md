# Asteroids
* Recreating the classic asteroids game as a way to learn OOP in a project.


## Running locally
This project uses uv as a package and project manager: https://github.com/astral-sh/uv .To run this project locally follow these steps:
* install uv
``` bash
pip install uv
```
* check if installed
``` bash
uv --version
```
* install python 3.13
``` bash
uv python install 3.13
```
* sync dependencies
``` bash
uv sync
```
* run
``` bash
uv run main.py
```

uv will run the game with the correct python version without you needing to specify the version or activating a venv.
